## Shifty 1.0.0-beta.1

### Added
- **In-app updates** — Check for updates from Settings → About; download from GitHub Releases
- **What's new** — Release notes in Settings and a first-run modal after updating
- **Release tooling** — `scripts/release.sh`, DMG + ZIP builds, and GitHub Release publishing

---

### Install (unsigned beta)

This build may not be notarized. macOS may block unsigned downloads.

1. Download **Shifty-1.0.0-beta.1-arm64.dmg** below
2. Open the DMG and drag **Shifty** to **Applications**
3. If macOS blocks the app, run in Terminal:
   ```bash
   xattr -cr /Applications/Shifty.app
   ```
4. Open Shifty normally

**Requires:** macOS on Apple Silicon (arm64)
